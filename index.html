Інтернет магазин

Цей проект має принцип SPA і написаний на фреймворку React, використовуючи менеджер станів Redux. Він включає безліч технологій, такі як: використання передпроцесора scss, Ajax, API запросы, хмарне сховище даний і фотографій - firebase, можливість аунтифікації через різні сервіси та багато інших. Даний інтернет магазин має безліч сторінок та компонентів, такі як: різні слайдери, посторінкова пагінація, фільтр та пошук товарів, кошик, оформлення замовлення, вікно авторизації, та ін. GitHub
